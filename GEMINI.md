# FitPulse - Workout Tracking App

## Project Overview
FitPulse is a mobile application built with Flutter that allows users to track their workouts, view history, and monitor progress.

## Tech Stack
- **Framework**: Flutter
- **UI Library**: flutter_shadcn_ui
- **State Management**: Provider
- **Navigation**: GoRouter

## Features
- User Authentication (Login/Signup)
- Dashboard with workout stats
- Workout logging and history
- Profile management
